Information Structure

Data set markup with XML language

The goal of this project is to develop a markup structure for a data set using XML language, in order to facilitate sharing and reuse of the data.

The data set I have chosen includes all major information one needs to know when he/she wants to buy a laptop computer, for example, the appearances, funtionalities, and prices of a few types of laptops.
